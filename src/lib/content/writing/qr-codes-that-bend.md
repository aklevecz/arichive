## The tension

Every beautiful QR code is one gradient away from being unscannable. The whole
craft lives on that knife's edge.

## The workflow

- Stable Diffusion in ComfyUI, ControlNets to anchor the code structure.
- IPAdapter to steer style without breaking the readable pattern.
- Endless parameter tinkering — the right balance shifts with every prompt.

## What I learned

- Which modules of a QR code you can sacrifice and which you can't.
- How error correction level buys you aesthetic headroom.
